## Jekyll and Github Pages
[Jekyll Instructions](https://docs.github.com/en/pages/setting-up-a-github-pages-site-with-jekyll)

```
$ jekyll new --skip-bundle .


$ bundle install


$ bundle exec jekyll serve --baseurl=""

```

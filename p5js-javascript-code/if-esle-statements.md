# If-Esle Statements

### Stacked conditional statement

```javascript
 if (label == "none") {
    currentState = none;
    // console.log("here");
  } else if (label == "O") {
    currentState = O;
  } else if (label == "R") {
    currentState = R;
  } else if (label == "A") {
    currentState = A;
  } else if (label == "N") {
    currentState = N;
  } else if (label == "G") {
    currentState = G;
  } else if (label == "E") {
    currentState = E;
  } else {
    currentState = none;
  }

```



### Nested conditional statement

```javascript
if ( myVariableName == "A" ) {
    if ( myVariableName == "B" ) {
        if ( myVariableName == "C" ) {
            // Run code here
        }
    }
}
```
